Getting Started

Welcome to the Student Performance Analysis System! To get started:

  1. Create an Academic Year
  2. Create a Class within the academic year
  3. Add Students to the class
  4. Define Chapters and their dependencies
  5. Enter student grades for each chapter
  6. View Recommendations for additional classes
System Overview

This system helps you track, analyze, and provide recommendations for additional mathematics classes based on students' performance across different chapters and their dependencies.

Performance categories:

  • Special Class (>90)
  • Unnecessary (80-90)
  • Required (70-80)
  • Very Necessary (<70)
